2. ssh 远程登录方式

使用ssh 公网服务器地址与对应frp端口号登录。公网服务器地址稍后公开;使用示例如下:

ssh node@192.168.58.194 -p 6002

3. 挂载共享文件夹

文件夹存放在node00节点中，其它节点可以使用如下的命令进行挂载，并直接访问node用户下的DataShare文件夹

sudo mount -t nfs node00-ubuntu:/home/node/DataShare /mnt/DataShare
